## 📝 产品信息

**产品名称**: CodeReview Agent
**标语**: AI-Powered CodeReview that tells you what's safe to ship
**一句话描述**: 🤖 AI 驱动的 CodeReview 智能体，帮助程序员自动识别哪些代码可提交、哪些需要人工审核。

## 🎯 Launch 详情

### 核心卖点（英文）
1. **Automatic Risk Detection** - AI detects security vulnerabilities, SQL injection, hardcoded keys
2. **Confidence Score** - Know exactly how risky your code is (0-100%)
3. **LangChain + LangGraph** - Modular and extensible
4. **6 LLM Providers** - OpenAI, Anthropic, 智谱AI, MiniMax, 阿里云, DeepSeek
5. **3 Ways to Use** - GitHub Action / Docker / CLI

### 吸引力语句
- "Stop wasting 2 hours on manual code review"
- "AI reviews every PR in 10 seconds"
- "Never miss a security vulnerability again"
